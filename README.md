# MY-TTGO-Watch
A GUI names hedge for the Lillygo products.
The supported devices are:
TTGO watch 2020 V1, V2 and V3, M5paper, M5core2, TTGO watch 2021 V1 and V2, TTGO watch 2020 and WT32-SC01 V3_2, V3_3 and the WT32-SC01.
The apps currently included are are:
Messages, weather, activity, media player, stopwatch, watchface manager, alarm, Sailing, gps status, astro, IR remote, Power-meter, Find phone, WiFi monitor, Calander, Cauculator, Weather Station, Tilt Mouse ,3D printer, Osm map, Osmand, MQTT player, MQQT conterol, Kodi Remote, ssid spam, ping, net tools, tic tac toe and pong.
Feel free to make new apps for the gui and let me know about it.
